# 题目描述

现在有 nnn 只水獭从低到高站成一排。Moca 希望每次能找到不低于某身高 hhh 的第一只水獭的编号（编号从 111 开始），请你编写一个程序完成这个任务。

# 输入格式

第一行两个正整数 nnn 和 mmm，其中 nnn 表示小水獭的个数，保证 1≤n≤5×1051 \le n \le 5 \times 10^51≤n≤5×105；mmm 表示接下来询问的次数，保证 1≤m≤5×1051 \le m \le 5 \times 10^51≤m≤5×105​。

接下来一行，输入 nnn 个正整数，第 iii 个正整数表示第 iii 只小水獭的身高 aia_iai​，保证 1≤ai≤1071 \le a_i \le 10^71≤ai​≤107，输入的 aia_iai​ 单调不减。

接下来 mmm 行，每行一个正整数 hih_ihi​，表示要查找的值，保证 1≤hi≤1071 \le h_i \le 10^71≤hi​≤107。

# 输出格式

输出 mmm 行，对每次询问，输出对应的不低于对应身高 hhh 的第一只水獭的编号；如果不存在这样的水獭输出 -1 。
