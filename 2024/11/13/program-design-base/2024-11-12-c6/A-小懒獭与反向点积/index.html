# 题目描述

小懒獭创造了一种独特的向量运算方式，她称之为 反向点积。反向点积是指将两个向量的元素按特定的方式反向相乘并求和。对于给定的两个长度相同的向量 AAA 和 BBB，小懒獭想知道它们的反向点积是多少。

具体来说，对于向量 A=(a0,a1,⋯ ,,an−1)A = (a_0, a_1, \cdots,, a_{n - 1})A=(a0​,a1​,⋯,,an−1​) 和 B=(b0,b1,⋯ ,bn−1)B = (b_0, b_1, \cdots, b_{n - 1})B=(b0​,b1​,⋯,bn−1​)，小懒獭的反向点积定义为：

A⊙B=a0×bn−1+a1×bn−2+⋯+an−1×b0A \odot B = a_0 \times b_{n - 1} + a_1 \times b_{n - 2} + \cdots + a_{n - 1} \times b_0 A⊙B=a0​×bn−1​+a1​×bn−2​+⋯+an−1​×b0​

每次给定两个长度相同的向量 AAA 和 BBB，请帮助小懒獭计算它们的反向点积。

# 输入格式

不定组数据输入，保证不超过 100010001000 组。

对于每组数据，第一行包含一个整数 nnn（1≤n≤10001 \le n \le 10001≤n≤1000），表示两个向量的长度；第二行包含 nnn 个整数 A0,A1,⋯ ,An−1A_0, A_1, \cdots ,A_{n − 1}A0​,A1​,⋯,An−1​，表示向量 AAA 的元素；第三行包含 nnn 个整数 B0,B1,⋯ ,Bn−1B_0, B_1 , \cdots , B_{n − 1}B0​,B1​,⋯,Bn−1​，表示向量 BBB 的元素。保证向量的每个元素的值在 [−100,100][−100,100][−100,100] 区间内。

# 输出格式

对于每组输入，输出一行一个整数，表示向量 AAA 和 BBB 的反向点积结果。
