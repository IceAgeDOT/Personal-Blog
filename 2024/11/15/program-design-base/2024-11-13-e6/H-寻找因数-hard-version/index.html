# 题目描述

给定 nnn 个数 a1,⋯ ,ana_1, \cdots, a_na1​,⋯,an​，求 m=a1×a2×⋯×anm = a_1 \times a_2 \times \cdots \times a_nm=a1​×a2​×⋯×an​ 的正因数个数。

# 输入

第一行一个正整数 ttt 代表数据组数，1≤t≤1001 \le t \le 1001≤t≤100。

对于每组数据，第一行一个正整数 nnn，代表数字个数，1≤n≤101 \le n \le 101≤n≤10。第二行 nnn 个正整数 aia_iai​，1≤ai≤1091 \le a_i \le 10^91≤ai​≤109。

# 输出

对于每组数据，输出一行一个正整数，代表 mmm 的不同正因数的个数，其中m=a1×a2×⋯×anm = a_1 \times a_2 \times \cdots \times a_nm=a1​×a2​×⋯×an​。

# 样例解释

对于第一组样例，6×2=126 \times 2 = 126×2=12，121212 的正因数有 1,2,3,4,6,121, 2, 3, 4, 6, 121,2,3,4,6,12 共 666 个。

对于第二组样例，171717 的因数有 1,171,171,17 共 222 个。

# 题解：线性筛 + 数论

考虑一个数的质因数分解：

m=p1c1p2c2⋯pkckm = p_1^{c_1} p_2^{c_2} \cdots p_k{c_k} m=p1c1​​p2c2​​⋯pk​ck​

其中 p1,⋯ ,pkp_1, \cdots, p_kp1​,⋯,pk​ 均为质数。我们知道，此时 mmm 的正因数个数为 (c1+1)(c2+1)⋯(ck+1)(c_1 + 1) (c_2 + 1) \cdots (c_k + 1)(c1​+1)(c2​+1)⋯(ck​+1)。

我们考虑将 a1,⋯ ,ana_1, \cdots, a_na1​,⋯,an​ 都进行质因数分解，再将同质数的质数相加，就得到了 mmm 的质因数分解。但是我们注意到，最大的质数还是 10910^9109 这个量级的，有什么办法吗？

不难发现，大于 109\sqrt {10^9}109​ 的质因数在每次分解 aia_iai​ 时至多只会出现一次，因此我们考虑将 109\sqrt {10^9}109​ 之前的质数用线性筛先筛出来，然后用它们对 aia_iai​ 进行分解，如果最后 aia_iai​ 不为 111，则剩下的必定是一个大于 109\sqrt {10^9}109​ 的质数，我们把它记下来即可。具体细节参考代码实现。
