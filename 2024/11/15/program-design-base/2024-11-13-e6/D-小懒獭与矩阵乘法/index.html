# 题目描述

小懒獭最近学习了矩阵乘法，她决定编写一个程序帮她求出两个矩阵的乘积。

已知两个矩阵 AAA 和 BBB 分别为 N×MN \times MN×M 和 M×PM \times PM×P 的矩阵，小懒獭需要计算出它们的乘积矩阵 CCC，其中 CCC 是一个 N×PN \times PN×P 的矩阵，满足：

C[i][j]=∑k=1MA[i][k]×B[k][j]C[i][j] = \sum_{k = 1}^M A[i][k] \times B[k][j] C[i][j]=k=1∑M​A[i][k]×B[k][j]

# 输入格式

第一行包含三个正整数 NNN、MMM、PPP，表示矩阵 AAA 的行数、列数和矩阵 BBB 的列数。

接下来 NNN 行，每行包含 MMM 个整数，表示矩阵 AAA 的元素。

接下来 MMM 行，每行包含 PPP 个整数，表示矩阵 BBB 的元素。

保证 1≤N,M,P≤1001 \le N, M, P \le 1001≤N,M,P≤100，矩阵中元素的值在 [−100,100][−100,100][−100,100] 范围内。

# 输出格式

输出 NNN 行，每行包含 PPP 个整数，表示矩阵乘积的结果；相邻整数间用空格隔开
