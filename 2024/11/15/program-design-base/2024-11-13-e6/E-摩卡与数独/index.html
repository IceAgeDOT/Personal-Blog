# 题目描述

Moca 最近发现了一种叫做数独的益智游戏。

经典的数独规则是这样的：

  • 将一个 9×99 \times 99×9 的方格分为 999 个 3×33 \times 33×3 的小方格，每个 3×33 \times 33×3 方格称为一个宫

  • 开始时一些方格中的数字是给定的，玩家需要根据推理，填写剩余方格中的数字，直到所有方格中都被填上了数字

  • 最后 在每一行、每一列、每一宫中，数字 1 - 9 都出现且只出现一次，则填写正确

Moca 喜欢做数独，但她不喜欢检查自己做的数独是否正确，尽管这很简单。所以，她希望你能帮她写一个程序，对于一个给定的初始状态，检查她做的数独是否正确。

# 输入

第一行一个正整数 ttt ，表示 Moca 做了多少数独，1≤t≤51 \le t \le 51≤t≤5。

接下来 ttt 组输入，每组输入之前有一个空行。每组输入 191919 行：前九行每行九个数字，为初始的数独局面，000 代表待填写的数字，其余数字代表初始给定的数字，保证初始局面中所有数字一定是 000 到 999 中的一个；接下来一行空行；后九行每行九个数字，表示对应做出来的数独，保证做出来的数独中的数字一定是 111 到 999 中的一个。

# 输出

输出 ttt 行，如果第 iii 组做出来的数独对于第 iii 组初始局面数独填写正确，那么对应输出行输出 Moca finish this sudoku perfectly! ；否则（包括做出来的数独虽然正确，但是不符合给定的初始局面的情况）输出 Moca is so careless! 。
