# 题目描述

给定两个长度相等的字符串 A,BA, BA,B，每个字符串只包含小写字母。字符串 AAA 和字符串 BBB 的距离定义为每一位字符的 ASCII\text {ASCII}ASCII 值差的绝对值之和。例如， A = 'abc' 和 B = 'bcd' ，则 AAA 和 BBB 的距离为：

∣a−b∣+∣b−c∣+∣c−d∣=1+1+1=3|a - b| + |b - c| + |c - d| = 1 + 1 + 1 = 3 ∣a−b∣+∣b−c∣+∣c−d∣=1+1+1=3

请编写一个程序，计算字符串 AAA 和 BBB 的距离。

# 输入格式

共两行，第一行代表字符串 AAA，第二行代表字符串 BBB。保证字符串只由小写字母组成，长度不超过 100010001000。

# 输出格式

一个正整数，表示字符串距离。
