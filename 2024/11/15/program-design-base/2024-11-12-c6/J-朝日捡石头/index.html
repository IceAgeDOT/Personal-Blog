# 题目描述

河边有 nnn 个石头，第 iii 个石头位置为 iii，价值为 aia_iai​。现在从这 nnn 个石头中选出 mmm 个石头，要求这 mmm 个石头中任意两个石头之间的距离不小于 kkk，最大化这 mmm 个石头的价值总和。

# 输入格式

第一行三个整数 n,m,kn, m, kn,m,k，表示石头数量，要选的石头数量，距离限制。1≤n≤3×105,1≤m,k≤n1 \le n \le 3 \times 10^5, 1 \le m, k \le n1≤n≤3×105,1≤m,k≤n。

第二行 nnn 个整数 aia_iai​，表示每个石头的价值。∣ai∣≤109|a_i| \le 10^9∣ai​∣≤109。

数据保证至少有一个选石头的方案。

# 输出格式

一行一个整数，表示答案。

# 题解：wqs 二分 + 动态规划

首先考虑暴力 dp\text {dp}dp，不难设计出 dp[i][j]dp[i][j]dp[i][j] 表示考虑前 iii 个石头，满足距离限制的条件下选出 jjj 个石头的最大价值，但是很显然时空双爆。

如果去掉第二维的数量限制，我们就能得到一个线性时间复杂度的一维 dp\text{dp}dp 方程：

dp[i]=max⁡(dp[i−1],dp[i−k]+a[i])dp[i] = \max (dp[i - 1], dp[i - k] + a[i]) dp[i]=max(dp[i−1],dp[i−k]+a[i])

于是我们自然而然地想到了用 wqs\text {wqs}wqs 二分来优化这个过程。下面证明答案具有凹凸性，即：

g(x)−g(x−1)≥g(x+1)−g(x)g(x) - g(x - 1) \ge g(x + 1) - g(x) g(x)−g(x−1)≥g(x+1)−g(x)

对 ∀ x∈[1,m−1]\forall \, x \in [1, m - 1]∀x∈[1,m−1] 恒成立。

显然对于一个合法的选取方式，从选出的石头中任意选择几个构成的子集，也一定合法。考虑 g(x−1)g(x - 1)g(x−1) 和 g(x+1)g(x + 1)g(x+1) 的这 2x2x2x 个石头，总能找到一种划分方案分出 222 组 xxx 个石头，其中我们总能挑出一组 xxx 个石头，使得 g(x−1)+g(x+1)≤2g(x)g(x - 1) + g(x + 1) \le 2 g(x)g(x−1)+g(x+1)≤2g(x) 成立，上凸性成立。
