# 一个问题

在思考这个问题之前，我们先来证几个定理：

定理 1： 设 u1,⋯ ,umu_1, \cdots, u_mu1​,⋯,um​ 是 nnn 维向量空间 FnF^nFn 中的 mmm 个向量，如果 m>nm > nm>n，则 u1,⋯ ,umu_1, \cdots, u_mu1​,⋯,um​ 线性相关。

证明： 考虑关于 FFF 中 mmm 个未知数 λ1,⋯ ,λm\lambda_1, \cdots, \lambda_mλ1​,⋯,λm​ 的方程

λ1u1+⋯+λmum=0(1)\lambda_1 u_1 + \cdots + \lambda_m u_m = 0 \tag {1} λ1​u1​+⋯+λm​um​=0(1)

对每个 1≤j≤m1 \le j \le m1≤j≤m，记 uj=(a1,j,⋯ ,an,j)u_j = (a_{1, j}, \cdots, a_{n, j})uj​=(a1,j​,⋯,an,j​).

则 (1)(1)(1) 成为线性方程组

{a1,1λ1+⋯+a1,mλm=0⋯an,1λ1+⋯+an,mλm=0(2)\begin {cases} a_{1, 1} \lambda_1 + \cdots + a_{1, m} \lambda_m = 0 \\ \cdots \\ a_{n, 1} \lambda_1 + \cdots + a_{n, m} \lambda_m = 0 \end {cases} \tag {2} ⎩⎨⎧​a1,1​λ1​+⋯+a1,m​λm​=0⋯an,1​λ1​+⋯+an,m​λm​=0​(2)

而对每个 1≤i≤n1 \le i \le n1≤i≤n，记 ei=(0,⋯ ,0,1,0,⋯ ,0)e_i = (0, \cdots, 0, 1, 0, \cdots, 0)ei​=(0,⋯,0,1,0,⋯,0) 表示第 iii 个分量为 111、其余分量为 000 的数组向量。显然易得这 nnn 个向量线性无关。
