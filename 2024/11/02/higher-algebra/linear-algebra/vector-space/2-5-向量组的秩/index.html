# 线性无关组

定义 1 极大线性无关组 ：设 SSS 是向量组。如果 SSS 的子集 M={α1,α2,⋯ ,αm}M = \{\alpha_1, \alpha_2, \cdots, \alpha_m\}M={α1​,α2​,⋯,αm​} 线性无关，并且将 SSS 中的任一向量 α\alphaα 添加进去构成的向量组线性相关，就称 MMM 是 SSS 的极大线性无关组。

引理 1 ：设 SSS 是向量组，MMM 是 SSS 的线性无关子集，则 MMM 是极大线性无关组 ⇔\Leftrightarrow⇔ SSS 中的任一向量都是 MMM 的线性组合。

证明 ：“⇒\Rightarrow⇒”：先设 MMM 是 SSS 的极大线性无关组。

设 α\alphaα 不属于 MMM，则 α1,α2,⋯ ,αm,α\alpha_1, \alpha_2, \cdots, \alpha_m, \alphaα1​,α2​,⋯,αm​,α 线性相关，数域 FFF 中存在不全为 000 的数 λ1,λ2,⋯ ,λm,λ\lambda_1, \lambda_2, \cdots, \lambda_m, \lambdaλ1​,λ2​,⋯,λm​,λ 使得：

# 秩

定义 2： 任一向量组 SSS 的任一极大线性无关组所含向量个数 rrr 称为向量组 SSS 的秩，记作 rank S\mathrm{rank} \, SrankS. 任一矩阵 AAA 的行向量组的秩称为这个矩阵的 行秩，AAA 的列向量组的秩成为 AAA 的 列秩。

推论 3： 如果向量组 S2S_2S2​ 是 S1S_1S1​ 的线性组合，则 rank S2≤rank S1\mathrm{rank} \, S_2 \le \mathrm{rank} \, S_1rankS2​≤rankS1​. 互为线性组合的向量组秩相等。

定义 3： 设 S1S_1S1​ 与 S2S_2S2​ 是同一个向量空间 VVV 中的两个向量组。如果 S1S_1S1​ 与 S2S_2S2​ 互为线性组合，就称 S1S_1S1​ 与 S2S_2S2​ 等价。如果矩阵 AAA 与 BBB 的行（列）向量组等价，就称 AAA 与 BBB 行（列）等价。

引理 3： 初等行变换不改变矩阵的行秩。

证明： 每次初等行变换前后的矩阵的行向量组等价。由等价的传递性知道：矩阵 AAA 经过若干次初等行变换得到的矩阵 BBB 的行向量组与 AAA 的行向量组等价。由此知：AAA 与 BBB 的行秩相等。

重点在于彼此互为线性组合。考察向量组之间的线性组合关系是本章关键。

# 用初等行变换计算秩

求线性方程组的秩： 将方程表成增广矩阵 MMM。方程组的秩就是 MMM 的行秩。用初等行变换化为行阶梯形，即可求得线性方程组的秩。

  2. 设复数域上的向量 α1,⋯ ,αn\alpha_1, \cdots, \alpha_nα1​,⋯,αn​ 线性无关。对复数 λ\lambdaλ 的不同值，判断向量组

    S:α1+λα2,⋯ ,αn−1+λαn,αn+λα1S: \alpha_1 + \lambda \alpha_2, \cdots, \alpha_{n - 1} + \lambda \alpha_n, \alpha_n + \lambda \alpha_1 S:α1​+λα2​,⋯,αn−1​+λαn​,αn​+λα1​
