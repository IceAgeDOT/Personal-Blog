# 原函数与不定积分的概念

定义 111：如果对 ∀ x∈I\forall \, x \in I∀x∈I，都有 F′(x)=f(x)F'(x) = f(x)F′(x)=f(x)，那么 F(x)F(x)F(x) 就称为 f(x)f(x)f(x) 在区间 III 上的一个 原函数。

注意：

  1. 原函数 不一定 存在。例如符号函数 f(x)=sgn(x)={1,x>00,x=0−1,x<0f(x) = \mathrm {sgn} (x) = \begin {cases} 1, & x > 0 \\ 0, & x = 0 \\ -1, & x < 0 \end {cases}f(x)=sgn(x)=⎩⎨⎧​1,0,−1,​x>0x=0x<0​，假设有原函数 F(x)F(x)F(x)，则：

    F(x)={x+C,x>0C,x=0−x+c,x<0F(x) = \begin {cases} x + C, & x > 0 \\ C, & x = 0 \\ -x + c, & x < 0 \end {cases} F(x)=⎩⎨⎧​x+C,C,−x+c,​x>0x=0x<0​

    但 F(x)F(x)F(x) 在 x=0x = 0x=0 处不可导，故假设错误，所以 f(x)f(x)f(x) 在 (−∞,+∞)(- \infty, + \infty)(−∞,+∞) 内不存在原函数。

    原函数存在定理：如果函数 f(x)f(x)f(x) 在区间 III 内连续，那么在区间 III 内存在可导函数 F(x)F(x)F(x)，使 ∀ x∈I\forall \, x \in I∀x∈I，都有 F′(x)=f(x)F'(x) = f(x)F′(x)=f(x)，即 连续函数一定有原函数。

  2. 原函数不唯一，若 F(x)F(x)F(x) 和 G(x)G(x)G(x) 都是 f(x)f(x)f(x) 的原函数，则 F(x)−G(x)=CF(x) - G(x) = CF(x)−G(x)=C，其中 CCC 为任意常数。

定义 222：在区间 III 上，函数 f(x)f(x)f(x) 的全体原函数称为 f(x)f(x)f(x) 在区间 III 内的 不定积分，记为 ∫f(x)dx\int f(x) \mathrm d x∫f(x)dx。

函数 f(x)f(x)f(x) 的原函数的图形称为 f(x)f(x)f(x) 的 积分曲线，求不定积分得到一 积分曲线族。

# 不定积分的性质

ddx[∫f(x)dx]=f(x),d[∫f(x)dx]=f(x)dx,∫F′(x)dx=F(x)+C,∫dF(x)=F(x)+C.\begin {matrix} \displaystyle \frac {\mathrm d} {\mathrm d x} \left[ \int f(x) \mathrm d x \right] = f(x), & \displaystyle \mathrm d \left[ \int f(x) \mathrm d x \right] = f(x) \mathrm d x, \\ \displaystyle \int F'(x) \mathrm d x = F(x) + C, & \displaystyle \int \mathrm d F(x) = F(x) + C. \end {matrix} dxd​[∫f(x)dx]=f(x),∫F′(x)dx=F(x)+C,​d[∫f(x)dx]=f(x)dx,∫dF(x)=F(x)+C.​

结论：微分运算与求不定积分的运算是 互逆 的。

先积后导全消掉，先导后积常数要。

定理 111：

∫[k1f(x)+k2g(x)]dx=k1∫f(x)dx+k2∫g(x)dx\int [k_1 f(x) + k_2 g(x)] \mathrm d x = k_1 \int f(x) \mathrm d x + k_2 \int g(x) \mathrm d x ∫[k1​f(x)+k2​g(x)]dx=k1​∫f(x)dx+k2​∫g(x)dx

证明：

[k1∫f(x)dx+k2∫g(x)dx]′=k1[∫f(x)dx]′+k2[∫g(x)dx]=k1f(x)+k2g(x)\begin {aligned} \left[ k_1 \int f(x) \mathrm d x + k_2 \int g(x) \mathrm d x \right]' & = k_1 \left[ \int f(x) \mathrm d x \right]' + k_2 \left[ \int g(x) \mathrm d x \right] \\ & = k_1 f(x) + k_2 g(x) \end {aligned} [k1​∫f(x)dx+k2​∫g(x)dx]′​=k1​[∫f(x)dx]′+k2​[∫g(x)dx]=k1​f(x)+k2​g(x)​

所以等式成立。

注意：

  1. 定理不包含 k1=k2=0k_1 = k_2 = 0k1​=k2​=0 的情形；

  2. 定理可以推广到有限多个函数线性组合的情形。
